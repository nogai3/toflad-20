Создаём окно

Для создания окна в TOFLAD-20 используется функция CreateWindow.Start, которая вызывается в файле main.py. Пример строки: 

CreateWindow.Start(GameTitle, 1280, 720, Colors.WHITE, 60)

Параметры функции:

  • Имя окна (title): Название окна, отображаемое в заголовке.
  • Ширина окна (width): Ширина окна в пикселях.
  • Высота окна (height): Высота окна в пикселях.
  • Цвет заливки (defaultColor): Цвет фона окна, например, Colors.WHITE.
  • FPS: Частота кадров в секунду.

Если нужно изменить поведение окна, откройте файл CreateWindow.py по пути util/window/CreateWindow.py.

Рисуем спрайт (draw)

Для рисования спрайтов используется файл sprites/Test.py. В функции draw определены элементы, такие как тело поезда, окна, колёса и дым. Для рисования используются функции из util/Drawer.py, такие как:

  • drawRect — рисует прямоугольники.
  • drawLine — рисует линии.
  • drawCircle — рисует круги.

Чтобы изменить спрайт, отредактируйте координаты и цвета в Test.py.

Загружаем спрайт (image)

Для загрузки изображений используется функция loadImage из файла util/Drawer.py. Она принимает путь к изображению. Чтобы отобразить изображение на экране, используйте функцию renderImage, передав окно, изображение и координаты.

Включаем музыку

Для воспроизведения музыки используется файл util/sound/Sound.py. Доступны функции:

  • play_music(path) — воспроизводит музыку из указанного файла.
  • play_sound(path) — воспроизводит звуковой эффект.

Убедитесь, что файлы находятся в поддерживаемом формате (например, MP3 или WAV).

Добавляем звуки

Для добавления звуков используйте функцию play_sound из Sound.py. Просто передайте путь к звуковому файлу, и он будет воспроизведён.

Ввод с клавиатуры

Для обработки ввода с клавиатуры используется файл util/input/KeyboardInput.py. Основные функции:

  • read() — считывает события клавиатуры.
  • update() — возвращает текущее состояние клавиш (нажаты, отпущены, удерживаются).
  • clear() — очищает состояние клавиш.

Пример использования можно найти в engine/controls/Controls.py.

Ввод с мыши

Для обработки ввода с мыши используется файл util/input/MouseInput.py. Основные функции:

  • read() — считывает события мыши (позиция, кнопки, колёсико).
  • update() — возвращает текущее состояние мыши.
  • clear() — очищает состояние мыши.

Пример использования также можно найти в engine/controls/Controls.py.

Работаем с логгером

Для работы с логами используется файл engine/logger/Logger.py. Основные функции:

  • create_log_file(filename, maintext) — создаёт лог-файл с указанным текстом.
  • write_in_log_file(filename, log_type, message, code) — записывает сообщение в лог.
  • read_log_file(filename) — читает содержимое лог-файла.
  • clear_log_file(filename) — очищает лог-файл.

Логи можно использовать для отладки и отслеживания событий.

Заключение

TOFLAD-20 — это удобный инструмент для создания окон, работы со спрайтами, звуками и ввода. Если у вас возникли вопросы, изучите исходный код или обратитесь к документации. Удачи в разработке!
